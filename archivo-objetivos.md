# Sistema de Gestión de Restaurantes

## Analisis del proyecto

### ¿Que problema soluciona?

-El sistema ayuda a solucionar la falta de organizacion del restaurante,
porque actualmente los pedidos se toman a mano y el control de ventas e 
inventario es muy malo, genera errores y desorden. El sistema permitira
registrar pedidos, emitir facturas y llevar control de ventas ya se a largo
o a corto plazo, y llevar control del inventario.
